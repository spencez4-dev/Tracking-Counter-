LOAD QUEST GARAGE — NET SCORE FIX

MINUS BUTTON BEHAVIOR
The minus button is now a true global correction.

Pressing minus subtracts one load from:
- Today's main load total
- Current hourly race progress
- Estimated work time
- Current hourly pace
- Best-hour calculations
- Daily goal progress
- Lifetime XP
- XP level progress
- Vehicle unlock requirements based on XP or level
- Hourly streak qualification

The current vehicle moves backward on the race track when a load is removed.

HISTORICAL RACE WINS
A previously completed race win remains in the permanent race-win record.
Minus does not erase a trophy that was already earned in an earlier completed moment.
However, the live vehicle and current-hour progress immediately move backward.
